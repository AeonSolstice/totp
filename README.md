# TOTP

This is a simple **t**ime-based **o**ne-**t**ime **p**assword implementation in the form of a web app. All calculations happen locally inside your browser, nothing is ever sent to a server. There is no tracking, analytics, or ads. All data is stored using your browser's storage system and can optionally be encrypted at rest.
